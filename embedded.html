Homemade Oscilloscope

The Homemade Oscilloscope is a C++ project with a Qt-based GUI :

  • Data is sent by a low-frequency generator through UART to a STM32,
  • The STM32 receives the data through UART, and sends it through a buffer to the computer,
  • The C++ App reads the data through the serial port, and processes it,
  • The GUI built with Qt allows to modify the display of the function originally sent,
  • You can display triangle functions, sine waves etc, as well as zoom in, change the scale or the color of the graph.
