Homemade Oscilloscope

The Homemade Oscilloscope is a C/C++ project with a Qt-based GUI :

  • Data is sent by a low-frequency generator through UART to a STM32,
  • The STM32 receives the data through UART, and sends it through a buffer to the computer,
  • The C++ App reads the data through the serial port, and processes it,
  • The GUI built with Qt allows to modify the display of the function originally sent,
  • The display features a zoom and scale slide, as well as a color change, run and reset button.

Technologies Used: C/C++, Qt

FPGA Chronometer

The FPGA chronometer possesses multiple features and is implemented on an Altera Cyclone V :

  • 4 digit ( up to 99.99 ) displayed with a 7 segments LED display,

MCP23S17 Driver

Implementation of an API to enslave an MCP23S17 I/O Expander through SPI :

  • Use a private SPI driver for STM32F3,
  • Allows multiple low-level functions as well as abstracted high level ones,
  • You can write/read registers, set/clear bits in a register at the lowest levels,
  • You can also use, inspired by arduino, digitalWrite and readBits of GPIOs at the higher level,
  • The interrupt controller is still to be implemented.

Technologies Used: C/C++, STM32

STM32 - RPi Communicatiion

UART Communication project between STM32 and RPi :

  • The STM32 broadcasts "TR" to notify transmission started,
  • The STM32 then broadcasts "AA" to start the data transmission and "ZZ" at the end of it,
  • The Raspberry Piwaits for "AA", then processes the data buffer until "ZZ" is received,
  • The process is repeated every 100 ms by interruption on Timer 6.

Technologies Used: C, Python, STM32, RPi

CPLD Calculator

The Calculator, implemented on a Xillinx CoolRunner II with several features :

  • Three digits are displayed with 7-segments LED displays, the operands and the result,
  • The operands can be increased from zero to nine (modulo nine) with the push buttons (one for each operand),
  • Addition, Multiplication and Substraction are available with three combinations.

Technologies Used: VHDL, FSM
