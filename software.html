Secret Santa

This Secret Santa pool generator features :

  • Generation of a secret santa pool with N people involved,
  • Allocation of each people to someone in the pool and sends the santa via email,
  • Use of IMAP to delete the last N emails sent by the used email for generation,
  • Use of Github Actions for CI and testing,
  • (WIP) GUI that eases the use of the app.

Technologies Used: Python, IMAP, SMTPLIB, SSL

Pathfinding 2D Matrix

Multiple Pathfinding algorithms solving a 2D Matrix with obstacles :

  • Generation of an environment (2D Matrix of L*H size) with a percentage of obstacle as parameter,
  • Generation of a path based on the chosen algorithm,
  • Pathfinding algorithms include : A*, Dijkstra, DFS, BFS, Spanning Tree Walk, Bi-directional A*.
  • (WIP) Github Actions Continuous Integration,
  • (WIP) Iterative Deeping A*, Best-First.

Technologies Used: Python, Pathfinding

Markov Decision Processes

Implementation of Markov Decision Processes in a 2D matrix :

  • Rewards for each tiles are set (only one environment possible),
  • Everytime the robot tries to move in one direction, there's a 10% chance that it goes left instead of straight, same for right,
  • Terminal nodes are set with rewards of -1 and +1,
  • The algorithm computes the utilities of each tiles until a certain condition,
  • At the end, the number of iterations to get the final utilities as well as the values of the utilities are displayed.

Technologies Used: Python, Non-Determinism

Maze Solver

The Maze Solver resolves an input maze in the .png format (black and white occupancy grid) :

  • The maze is generated by the generator.cpp code provided by Jacek Wieczorek,
  • Three algorithms are provided : cell, line and corridor,
  • Cell solves cell by cell and find the shortest path with a "high" computation time,
  • Line solves the maze by going as far as possible in a direction to find a child, returning a mostly longer path, but with "lower" computation time,
  • (WIP) Corridor keeps going while it doesn't hit a corner, resulting in very low computation time and better path than the line solver,
  • The paths are displayed in color on the png image of the maze and saved as a new png file.

Technologies Used: Python, Pathfinding, OpenCV
